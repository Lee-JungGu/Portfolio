About Portfolio Project

  그동안 작업한 프로젝트 정보를 제공할 수 있는 사이트를 반응형으로 제작하였습니다. 정보 전달에 적합한 구조와 디자인을 고민하였고 효율적인 작업을 위해 구조를 모듈화하여 작업하였습니다. 데이 나이트 모드를 구현하면서 localstorage를 공부할 수 있었고 메소드들도 객체화하여 코드를 한눈에 볼 수 있도록 하였습니다.
