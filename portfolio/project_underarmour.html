About Under Armour Project

  스포츠 브랜드 언더아머 사이트를 리뉴얼하여 디자인하였습니다. 메인 페이지와 3개의 서브 페이지로 구성되어 있습니다. 기획 디자인은 8월에 완료하였고 구현 작업은 12월부터 시작하여 현재 진행 중입니다. 메인 페이지 코딩까지 완료하였고 나머지 3개의 페이지는 추후 구현할 예정입니다.
